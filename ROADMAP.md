# Roadmap

## 长期维护

- prompt 模板与 mdParser 解析规则的耦合迭代（formatSpec.js 单一来源）
- 不同 LLM 的 prompt 适配（Gemini 引用标记等）
- LaTeX / Markdown 渲染兼容性

## 功能拓展

按需求驱动，不做预设排期：

- 日历视图 / 复习回顾
- 环状进度图 / 学习计划
- 批量管理章节目录（目前只有单张卡片粒度）
- 三点菜单增强（分享）
- 跨设备同步（iCloud / Git / Supabase）
- FSRS 算法升级（等 SM-2 不够用时）
- 导入去重增强（模糊匹配）
- 手势交互打磨（左滑删除、长按拖拽排序、手势翻卡等，实际用一段时间后按痛点优先级做）

## 衍生项目

**刷题模式**：复用 Mnemos UI 框架，数据模型替换为题目 + 选项 + 解析，追踪正确率而非 SM-2 间隔。同一范式：prompt 生成题库 .md → app 消费。待评估是 fork 还是做成 Mnemos 第二模式。
